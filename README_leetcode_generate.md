# leetcode_generate Usage

## Preparements:

```cmd
pip install requests
pip install pyquery
```

## Config:

Edit your own username, password, language and repo in the **config.cfg** file

```
[leetcode]

username = username
password = password
language = python
repo = https://github.com/bonfy/leetcode
```

## Run

```python
python3 leetcode_generate.py
```
